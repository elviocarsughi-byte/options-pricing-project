# Options Pricing & Derivatives Project

This project was completed as part of the MSc Finance with Data Science at UCL.

## Overview
The project analyses option pricing and risk using both theoretical and empirical approaches, including:

- Binomial option pricing models
- Put-call parity and arbitrage strategies
- Replicating portfolios
- Empirical option pricing using real market data (AMZN)
- Monte Carlo simulation for barrier options
- Variance reduction techniques (control variates)

## Key Results
- Built a 3-period binomial tree to price European options
- Compared theoretical prices with real market data
- Simulated 10,000 GBM paths to price barrier options
- Achieved ~85% variance reduction using control variates

## Tools Used
- Python (NumPy, pandas)
- Monte Carlo simulation
- Financial modelling techniques
